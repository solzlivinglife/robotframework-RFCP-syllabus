# 2 Getting Started with Robot Framework

This chapter introduces participants to the foundational concepts of [Robot Framework](../glossary#robot-framework).
It covers the basics of how automation specifications are structured, how [suites](../glossary#suite) are organized, and the execution process.
Participants will learn how Robot Framework is run and explore the generated reports and logs that document test results.

The chapter also provides an overview of [suite](../glossary#suite) structures,
the role of libraries and [resource files](../glossary#resource-file),
and how to import them.
Additionally, it delves into the core syntax of Robot Framework,
focusing on how [keywords](../glossary#keyword) are defined and used, the types of [keyword](../glossary#keyword) [arguments](../glossary#argument),
and how keyword documentation is interpreted to ensure clarity and maintainability.
