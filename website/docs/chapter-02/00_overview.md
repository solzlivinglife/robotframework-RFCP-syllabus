# 2 Getting Started with Robot Framework

This chapter introduces participants to the foundational concepts of Robot Framework.
It covers the basics of how automation specifications are structured, how suites are organized, and the execution process.
Participants will learn how Robot Framework is run and explore the generated reports and logs that document :term[test]{term="Test Case"} results.

The chapter also provides an overview of suite structures,
the role of libraries and :term[resource files]{term="Resource File"},
and how to import them.
Additionally, it delves into the core syntax of Robot Framework,
focusing on how :term[keywords]{term="Keyword"} are defined and used, the types of :term[keyword]{term="Keyword"} :term[arguments]{term="Argument"},
and how :term[keyword]{term="Keyword"} documentation is interpreted to ensure clarity and maintainability.
